### Установка:
Выполните поочередно следующие команды в корневой директории проекта:
1. Синхронизация зависимостей и настройка окружения
```bash
make install
```
2. Сборка пакета
```bash
make build
```
3. Установка игр в систему как глобальных утилиты
```bash
make package-install
```

### Запуск игр:
```bash
brain-even         # Игра «Четное/Нечетное»
brain-calc         # Игра «Калькулятор»
brain-gcd          # Игра «Наибольший общий делитель»
brain-progression  # Игра «Арифметическая прогрессия»
brain-prime        # Игра «Простое число»
```
